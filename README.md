# agustin-gonzatto-porfolio
--- 

Personal portfolio website built with **Angular** to present my professional profile and CV in a clear and modern way.

This project serves as an online resume, showcasing my experience, skills, and selected projects for recruiters and hiring teams.

---

## About

The goal of this portfolio is to provide a concise overview of my background through a responsive and easy-to-navigate web interface.

The project will evolve over time as new features and content are added.

---

## Tech Stack

- Angular
- TypeScript
- HTML5 / CSS3
